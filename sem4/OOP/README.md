# 🎯 Course Overview

Object Oriented Programming (OOP) introduces the principles of software development using object-oriented design and programming concepts. The course focuses on creating modular, reusable, and maintainable software through the application of classes, objects, inheritance, polymorphism, encapsulation, and abstraction.

Throughout the semester, our team developed a **Booking Management System** using Java and the Vector data structure to manage and organize booking records efficiently.

---

# 🚀 Main Project: Booking Management System

## Project Overview

The Booking Management System was developed to manage customer booking information and streamline reservation processes through a user-friendly Java application.

The project aimed to apply object-oriented programming principles while implementing efficient data storage and retrieval using Java's Vector collection framework.

---

# 📂 System Features

### Customer Management

- Add customer records
- Update customer information
- Delete customer records
- Search customer details

### Booking Management

- Create new bookings
- Modify booking details

### Data Management

- Store records using Java Vector
- Dynamic data storage
- Efficient record retrieval
- Data validation

# 🏗️ OOP Concepts Applied

## Encapsulation

Data and methods were bundled within classes to ensure secure access and better code organization.

### Example

- Customer Class
- Booking Class
- Management Class

---

## Inheritance

Inheritance was used to promote code reusability and establish relationships between related classes.

### Benefits

- Reduced code duplication
- Better maintainability
- Improved system scalability

---

## Polymorphism

Different methods were implemented to perform similar operations based on object behavior.

### Benefits

- Flexible program design
- Improved extensibility
- Simplified maintenance

---

## Abstraction

Complex implementation details were hidden while exposing only necessary functionalities to users.

### Benefits

- Cleaner system architecture
- Improved user interaction
- Better software design

---

# 💡 Reflection

Object Oriented Programming was one of the most important programming courses in my degree because it introduced me to modern software development principles that are widely used in industry.

Through the development of the Booking Management System, I gained practical experience in designing software using classes, objects, and object-oriented concepts rather than relying solely on procedural programming approaches. The project helped me understand how large applications can be organized into smaller, reusable, and maintainable components.

One of the most valuable lessons learned from this course was the importance of encapsulation, inheritance, polymorphism, and abstraction in creating scalable software solutions. Applying these concepts allowed me to design cleaner code structures and improve program maintainability.

The project also strengthened my Java programming skills and introduced me to the Java Collections Framework, particularly the use of Vector for managing dynamic data. Working with booking records improved my understanding of data organization, object relationships, and program logic.

Additionally, the project enhanced my problem-solving abilities, debugging skills, and software design thinking. Collaborating with team members also improved my communication and teamwork skills during system development.

Overall, this course provided a strong programming foundation and changed the way I approach software development. The knowledge gained from OOP continues to be useful in many other courses involving application development, software engineering, database systems, and enterprise applications.
