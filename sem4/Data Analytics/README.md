Throughout the semester, our team developed a project analyzing vehicle registration and fuel consumption datasets to investigate CO₂ emission patterns across different fuel types and Malaysian states.

# 🚀 Main Project: Vehicle CO₂ Emissions Analytics

## Project Overview

The project combined vehicle registration data with fuel consumption and emissions datasets to analyze environmental trends and identify factors influencing CO₂ emissions.

The objective was to prepare, clean, analyze, and visualize large datasets while applying machine learning techniques to classify emission levels across different states.

# 📂 Phase 1: Data Preparation & Cleaning

### Activities Performed

- Loaded multiple datasets using Pandas.
- Standardized vehicle maker and model names.
- Merged vehicle registration and fuel consumption datasets.
- Identified unmatched records.
- Removed missing CO₂ emission values.
- Eliminated duplicate records.
- Generated summary statistics and aggregated datasets.

### Key Findings

- Total vehicle registrations analyzed: 263,578
- Successfully matched records: 9,212
- Unmatched records: 251,480
- Petrol vehicles showed lower average emissions compared to diesel vehicles.
- Significant emission variations were observed across different states.

# 📊 Phase 2: Data Analytics & Visualization

### Exploratory Data Analysis (EDA)

The cleaned dataset was analyzed to identify relationships between fuel type, state, and CO₂ emissions.

### Analytics Performed

- Average CO₂ emissions by fuel type.
- Average CO₂ emissions by state.
- Multi-index and pivot table analysis.
- Top 10 and Bottom 10 state emission comparisons.
- Emission category classification (Low, Medium, High).

### Machine Learning Models

#### K-Nearest Neighbors (KNN)

- Trained using state-based emission categories.
- Achieved excellent classification performance on the test dataset.

#### Support Vector Machine (SVM)

- Applied for emission category prediction.
- Produced high classification accuracy and strong predictive performance.

### Visualization Techniques

- Bar Charts
- Comparative Analysis Charts
- Emission Ranking Visualizations
- State-based Emission Distribution Analysis

### Tools & Libraries Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
