# PROGRAMMING TECHNIQUE I (C++)

## Course Information

**Course:** Programming Technique I  
**Semester:** Semester 1  
**Programme:** Bachelor of Computer Science (Data Engineering)

---

# Course Overview

Programming Technique I introduces the fundamental concepts of computer programming using C++. The course focuses on problem-solving techniques, algorithm development, program design, and implementation through coding.

Throughout the semester, I learned how to write structured programs, apply logical thinking, and develop solutions to computational problems using C++.

---

# Assignments

## 1. Basic Input and Output

### Description
Development of simple programs using input, output, variables, and data types.

### Learning Outcomes
- Understand C++ syntax.
- Use variables and constants.
- Perform input and output operations.
- Apply basic arithmetic calculations.

---

## 2. Selection Structures

### Description
Implementation of decision-making statements such as if, if-else, and switch.

### Learning Outcomes
- Apply conditional logic.
- Design programs with multiple decision paths.
- Improve logical problem-solving skills.

---

## 3. Repetition Structures

### Description
Development of programs using loops such as for, while, and do-while.

### Learning Outcomes
- Automate repetitive tasks.
- Control program execution efficiently.
- Solve iterative problems.

---

## 4. Functions and Modular Programming

### Description
Creation of reusable functions to improve program organization and readability.

### Learning Outcomes
- Understand function declaration and definition.
- Apply parameter passing.
- Improve code modularity and maintenance.

---

## 5. Arrays and Data Processing

### Description
Implementation of arrays to store and manipulate collections of data.

### Learning Outcomes
- Store multiple values efficiently.
- Perform searching and processing operations.
- Develop structured data handling skills.
