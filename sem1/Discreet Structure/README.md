# Course Overview

Discrete Structure is a fundamental course in computer science that introduces mathematical concepts used in computing and problem-solving. The course covers logic, sets, relations, functions, counting techniques, proof methods, graph theory, trees, and discrete probability.

The assignments and exercises in this folder demonstrate the application of mathematical reasoning and analytical thinking to solve computational problems.

---

# Assignments

## 1. Propositional Logic and Truth Tables

### Description
Construction and analysis of logical statements using truth tables and logical operators.

### Learning Outcomes
- Understand logical reasoning.
- Construct truth tables.
- Evaluate logical equivalence.
- Apply logical operations in problem solving.

---

## 2. Sets, Relations and Functions

### Description
Study of set operations, relations between elements, and mathematical functions.

### Learning Outcomes
- Perform set operations.
- Analyze relations and properties.
- Understand mappings and functions.
- Apply mathematical concepts to computing problems.

---

## 3. Counting Principles and Combinatorics

### Description
Application of permutations, combinations, and counting techniques.

### Learning Outcomes
- Solve counting problems.
- Apply combinatorial reasoning.
- Analyze probability-related scenarios.
- Develop structured problem-solving approaches.

---

## 4. Graph Theory and Trees

### Description
Analysis of graphs, trees, paths, circuits, and network structures.

### Learning Outcomes
- Understand graph representations.
- Analyze connectivity and traversal.
- Apply tree structures in computing.
- Understand network modelling concepts.
