# Course Overview

Digital Logic introduces the fundamental concepts of digital systems and electronic circuit design. The course covers number systems, logic gates, Boolean algebra, combinational circuits, sequential circuits, flip-flops, counters, multiplexers, and digital circuit implementation.

The assignments and laboratory exercises in this folder demonstrate the practical application of digital logic concepts through circuit design, analysis, and simulation.

---

# Assignments

## 1. JK Flip-Flop

### Description
Design and analysis of JK Flip-Flop circuits including truth tables, state transitions, and timing behavior.

### Learning Outcomes
- Understand sequential logic circuits.
- Analyze state changes based on clock inputs.
- Apply excitation and characteristic tables.
- Design memory elements using flip-flops.

---

## 2. JK Flip-Flop 4-Bit Counter

### Description
Implementation of a 4-bit counter using JK Flip-Flops.

### Learning Outcomes
- Design synchronous and asynchronous counters.
- Understand binary counting mechanisms.
- Analyze clock pulse propagation.
- Implement sequential circuit designs.

---

## 3. Multiplexer and Demultiplexer

### Description
Design and analysis of Multiplexer (MUX) and Demultiplexer (DEMUX) circuits.

### Learning Outcomes
- Understand data selection techniques.
- Apply multiplexing concepts in digital communication.
- Design combinational circuits using logic gates.
- Simplify circuit implementation.

---

## 4. Deeds Assignment

### Description
Application of digital logic concepts to solve logic design problems and circuit implementation tasks.

### Learning Outcomes
- Improve logical thinking skills.
- Apply Boolean algebra for simplification.
- Design efficient digital circuits.
- Interpret digital system requirements.

Initially, I found state transitions confusing, but after repeatedly analyzing truth tables and timing diagrams, I became more confident in understanding sequential logic behavior.

### What I Gained

### What I Gained
