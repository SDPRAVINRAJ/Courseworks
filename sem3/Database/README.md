# 🎯 Course Overview

This course focuses on database design, development, and management. Students learn how to model data, design relational databases, implement SQL queries, and ensure data integrity.

The main project for this course was the development of the **KADA Loan Application System** database.

---

# 📂 Project: KADA Loan Application System

## Description

The KADA Loan Application System was developed to manage loan applications submitted by users while allowing administrators to review, approve, and manage applications efficiently.

---

## Key Learning Areas

- Entity Relationship Diagram (ERD)
- Database Design
- Normalization
- SQL Queries
- Primary and Foreign Keys
- Data Integrity
- CRUD Operations

---

## Reflection

Through this project, I learned how proper database design affects system performance and data consistency. Designing the database schema helped me understand relationships between entities and how data flows throughout a system.
