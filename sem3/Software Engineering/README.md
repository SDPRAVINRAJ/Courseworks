# 🎯 Course Overview

Software Engineering focuses on applying engineering principles to software development. The course covers requirements analysis, software design, project management, testing, and quality assurance.

The primary project was the development of the **KADA Loan Application System**.

---

# 📂 Project: KADA Loan Application System

## Description

The KADA Loan Application System was developed to digitalize the loan application process and improve efficiency for both applicants and administrators.
