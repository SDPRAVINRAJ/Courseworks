# 🎯 Course Overview

System Development Technology focuses on developing web-based systems using modern development practices and architectural patterns.

The primary project completed was the **KADA Loan Application System**, developed using the MVC architecture.

---

# 📂 Project: KADA Loan Application System

## Description

The KADA Loan Application System enables users to submit loan applications online while allowing administrators to review, approve, and manage applications.

- System Integration
